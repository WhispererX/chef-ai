# chef-ai
Chef AI is a personal cookbook app that lets you store your own recipes, manage your pantry, and cook with an AI chef that understands your ingredients and guides you step by step.

## Setup

This is a React Native Expo app built with JavaScript.

### Prerequisites
- Node.js (v20 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

### Running the App

- Start the development server:
```bash
npm start
```

- Run on Android:
```bash
npm run android
```

- Run on iOS:
```bash
npm run ios
```

- Run on Web:
```bash
npm run web
```
